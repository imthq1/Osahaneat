# OsahanEat Food Backend API

## Giới thiệu
OsahanEat Food Backend API là hệ thống backend cho ứng dụng đặt đồ ăn trực tuyến. API này cung cấp các endpoint để quản lý nhà hàng, món ăn, đơn hàng và người dùng.

## Công nghệ sử dụng
- Java
- Spring Boot
- Spring Data JPA
- Spring Security 6
- JSON Web Tokens (JWT)
- MySQL
- Maven
- ...............
